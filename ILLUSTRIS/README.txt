To create the data base, run the following commands in terminal!
	python CREATE_HALOS.py
	python CREATE_SUBHALOS.py
	python FIND_SATELLITES.py
	python FIND_RANGES.py
This will create the corresponding tables in a database titled "halos_and_subhalos.db"
	
To create velocity function plots, run the "velocity_functions.py" file.
To delete all tables from the database, run the "DROP_TABLES" file.
